## 📌 What is Business Intelligence?

The goal of Business Intelligence is to make better decisions by using Data.

### 7 Stages of BI:
1. Understanding the Problem Statement
2. Collecting and Understanding the Data
3. Data Cleaning and EDA (Exploratory Data Analysis)
4. Enhancing the Data
5. Data Analysis
6. Data Visualization
7. Communication with Stakeholders

### BI Use Cases:
- Telecom companies use BI tools to identify network issues in Real Time
- Businesses use Dashboards to track Inventory Management and Order Fulfilment efficiency

### BI Tools in the Market:
Tableau | Qlik | Power BI | Zoho Analytics | Sisense

---

## ⚡ What is Power BI?

- A Data Visualization and Business Intelligence tool by Microsoft
- Source data from multiple platforms and build Reports & Dashboards
- One of the Market Leaders in BI

### Power BI has 3 Tools:

| Tool | Description |
|------|-------------|
| 🖥️ Power BI Desktop | Windows app to source data and create Reports |
| ☁️ Power BI Service | Cloud-based SaaS — access at app.powerbi.com |
| 📱 Power BI Mobile | Mobile app to consume Reports on the go |

---

## 🔄 Power BI in the Data Analysis Lifecycle

| Phase | Description |
|-------|-------------|
| Data Collection | Collect data from various sources |
| Data Cleaning | Apply transformations, standardize the data |
| Data Modelling | Organize data from multiple sources into Relationships & Tables |
| Data Visualization | Create charts/graphs to identify Insights & Trends |
| Communication | Effective storytelling with data |

---

## 🔁 Flow of Data in Power BI

Create Report → Get Data → Transform Data → Create Model → Add Visualizations → Add Pages → Dashboard → Publish 🚀

### Architecture:
- **Power BI Desktop** — Extract, Transform, Load, Build Reports
- **Power BI Service** — Publish and Share
- **Power BI Mobile** — Access and Consume

## 🏢 Real-World Use Case — Swiggy

Using Power BI dashboards, Swiggy can track:
1. % of Orders Delivered on Time
2. Number of Orders in Last period
3. Most Ordered Dish in a State
4. Peak Timing of Orders
5. Which customers are frequently ordering
6. Pros and Cons of past decisions
7. Congested Delivery areas to improve service

## 🖥️ Key Components in Power BI Desktop

| Component | Description |
|-----------|-------------|
| Report | Collection of Visuals, Charts and Tables built on a Dataset |
| Extract | Extracting data from multiple sources |
| Transform | Cleaning and Merging data before analysis |
| Load | Transformed data used for building Reports |
| Model | Defines Relationships between multiple Tables |
| Filters | Select a subset of data based on conditions |
| Visualization | Charts/Graphs to identify Patterns & Insights |
| Format Report | Colors and Fonts to make it visually appealing |
| Tiles | A single Visual or Table pinned in the Dashboard |
